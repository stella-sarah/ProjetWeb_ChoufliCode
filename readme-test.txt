# User Management Module - Projet Web MVC

## 📋 Description
Module de gestion des utilisateurs pour une application web, développé en **architecture MVC** (Modèle-Vue-Contrôleur).  
Cette partie inclut :
- **FrontOffice** : Inscription, connexion et gestion de profil utilisateur
- **BackOffice** : Tableau de bord admin avec gestion des rôles et employés

## ✨ Fonctionnalités clés
- ✅ **Inscription utilisateur** avec validation
- 🔐 **Connexion sécurisée** 
- 📝 **Modification du profil** utilisateur
- 👨💻 **Interface admin** avec :
  - Liste des employés
  - Gestion des rôles (admin/employé)
  - Tableau de bord statistique
- 🎨 **Design responsive** avec 2 thèmes :
  - `style.css` pour le FrontOffice
  - `styleback.css` pour le BackOffice
